# TR Rendimiento — Firestore (Vercel Ready)

App de rendimiento por dependientx basada en el patrón TR-Lista.

## Deploy
1. Subir a GitHub
2. Importar en Vercel
3. Configurar variables de entorno
4. Listo

## Variables requeridas
GOOGLE_SHEETS_API_KEY
GOOGLE_SHEETS_ID

FIREBASE_API_KEY
FIREBASE_AUTH_DOMAIN
FIREBASE_PROJECT_ID
FIREBASE_STORAGE_BUCKET
FIREBASE_MESSAGING_SENDER_ID
FIREBASE_APP_ID
